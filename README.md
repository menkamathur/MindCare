# MINDCARE--digital-mental-health-system
 A mental health support web app
 
A digital platform to support mental wellness through mood tracking,
mental health assessment, and personalized suggestions.

✨ Features:
- 🔐 User Login & Registration
- 📋 Mental Health Onboarding Quiz
- 📊 Mood Tracker with Dashboard
- 💬 AI Chatbot for Support
- 📈 Wellness Progress Charts
